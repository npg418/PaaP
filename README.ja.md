# PDF as a Page for Manaba

[English](./README.md) | 日本語

ManabaのPDFを見ようとするとダウンロードされちゃうの、辟易してませんか?  
_Downloadsフォルダをきれいに保ちましょう！_  
この拡張機能は、Manaba内でPDFファイルをページとして表示することができちゃうんです。全部ブラウザ内で完結させたい人におすすめです。

## インストール方法

### Firefox系列の場合

1. Firefoxアドオンページにアクセス: <https://addons.mozilla.org/ja/firefox/addon/paap-for-manaba/>
2. 「Firefoxへ追加」をクリックして、インストールのプロンプトに従う。
3. **ウザいPDFのダウンロードにさようなら！**

### Chromium系列の場合

1. リリースページにアクセス: <https://github.com/npg418/PaaP/releases>
2. 最新の`PaaP.zip`ファイルをダウンロード。
3. ファイルを解凍して`PaaP`フォルダにする。
4. Chromeを開いて`chrome://extensions/`にアクセス。
5. 「デベロッパーモード」を有効にする（右上のトグル）。
6. 「パッケージ化されていない拡張機能を読み込む」をクリックして、`PaaP`フォルダを選択。
7. **ウザいPDFのダウンロードにさようなら！**
