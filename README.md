# PDF as a Page for Manaba

English | [日本語](./README.ja.md)

Tired of PDFs being downloaded when you try to view them on Manaba?  
_Let's keep your Downloads folder clean!_  
This extension allows you to view PDF files as a page within Manaba. It's perfect for those who want to keep everything within the browser.

## Installation

### For Firefox-based browsers

1. Go to the Firefox Add-ons page: <https://addons.mozilla.org/en-US/firefox/addon/paap-for-manaba/>
2. Click "Add to Firefox" and follow the prompts to install the extension.
3. **Say goodbye to annoying PDF downloads!**

### For Chromium-based browsers

1. Go to the release page: <https://github.com/npg418/PaaP/releases>
2. Download the latest `PaaP.zip` file.
3. Extract the file to create a `PaaP` folder.
4. Open Chrome and navigate to `chrome://extensions/`.
5. Enable "Developer mode" (toggle in the top right corner).
6. Click "Load unpacked" and select the `PaaP` folder.
7. **Say goodbye to annoying PDF downloads!**
